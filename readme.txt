Course Files Block for Moodle

Provides display of all files in the current course from the course page.

To install, place all files in /blocks/course_files and visit /admin/index.php in your browser.

This block has been tested on the following versions of Moodle: 2.5.

How it works

This block is used to display all the files of the current course.
When the block is added successfully, there will be a link in the block linking to a page displaying all the files of the course.
This blcok can only be added / configure in course page.
This block can only be added / configure by the manager of the course (teacher).
